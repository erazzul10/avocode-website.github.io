Basics of the Avocode app

Avocode’s UI is split into three main sections. A panel on the left, a large canvas to preview your designs in the middle, and inspector panel on the right for details about selected layers.

Get code from layer styles

Export CSS, Swift, Android & React Native code from the Styles panel on your right.

Use the Ruler Tool to measure distances

Just hold ALT to start using the Ruler Tool.

Select any layer and hit “Export”

Choose if you want to export PNG, JPEG, SVG, or WebP.

You can also export the assets in various sizes for iOS and Android like @2x, @3x, xhdpi etc.

Tip: You can select more layers at once by holding SHIFT
